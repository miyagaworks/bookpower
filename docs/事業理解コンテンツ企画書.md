# 事業理解コンテンツ企画書

## 📌 企画概要

### 目的
- 本を書くことで「事業の本質」が明確になるプロセスを示す
- 「なんとなく」が「明確」に変わる感覚を体験してもらう
- 事業理解が深まることで得られる具体的な成果を伝える

### ターゲット
- 自分の事業を30秒で説明できない経営者
- 理念が社員に伝わらないと悩んでいる経営者
- 事業の方向性が曖昧だと感じている経営者
- 次のステージへの道筋が見えない経営者

---

## 📖 コンテンツ構成案

### セクション1: オープニング - 30秒で説明できますか？
**見出し**: 「なんとなく」が「明確」に変わる瞬間

**内容**:
- **問いかけ**: 「あなたの事業について、30秒で説明してください」
  - 多くの経営者が詰まってしまう
  - 「えーっと、うちは……」「簡単に言うと……いや、でも……」
  - 頭の中では理解しているはずなのに、うまく言葉にできない

- **真実の提示**:
  - これはあなたの説明力が低いからではない
  - 事業の本質が、まだ言語化されていないだけ
  - 本を書くことは、自分の事業を徹底的に言語化すること

- **言語化する項目**:
  - なぜこの事業を始めたのか
  - どんな問題を解決しているのか
  - 誰のためのサービスなのか
  - 他社との違いは何なのか
  - 自分の強みは何なのか

- **起こる変化**:
  - モヤモヤしていたものが、クリアになる
  - 曖昧だったものが、はっきりする
  - バラバラだったものが、一つにまとまる

**ビジュアル要素**:
- モヤモヤ → クリアへの変化を示すイラスト
- 言語化する5つの項目をアイコン化

### セクション2: 実例 - 20年目の気づき
**見出し**: 飲食店経営者が初めて知った「本当に提供していたもの」

**内容**:
- **ストーリー**: 飲食店経営者（20年）の体験談
  - 本を書く前: 「美味しい料理を出す店」だと思っていた
  - 言語化の過程で気づいた: 提供していたのは「料理」じゃなく「居場所」
  - 常連客が求めていたのは「ほっとできる場所」だった

- **具体的な変化**:
  - メニューをシンプルに: 30種類 → 15種類
  - レイアウト見直し: カウンター席の間隔を広げる
  - BGMの変更: 落ち着いた雰囲気に
  - スタッフ教育: 「料理を早く出す」→「お客様が心地よく過ごせる」

- **結果**:
  - 客単価は変わらないのに、リピート率40% → 75%
  - 売上1年で1.5倍
  - **これは、本を書かなければ決して起きなかった変化**

**ビジュアル要素**:
- ビフォー・アフターの比較図
- 数字の変化（40% → 75%）を視覚的に強調

### セクション3: 説明力が格段に向上する
**見出し**: 本を書く前と後で、最も大きく変わるもの

**内容**:

#### Before: うまく説明できない…
「うちのサービスは、えーっと、要するに、お客様の課題を解決するための、まあ、総合的なソリューションというか……」

特徴:
- 長い
- 曖昧
- 相手は「？」という表情

#### After: 30秒で相手が理解してくれる！
「中小企業の採用課題を、3ヶ月で解決するサービスです。独自の手法で、『辞めない社員』を採用できるようになります」

特徴:
- 短い
- 明確
- 相手は「なるほど！」という表情

#### なぜこの変化が起こるのか
- 本を書くプロセス = 何度も何度も推敲するプロセス
- 「この表現で伝わるだろうか？」を繰り返す
- **本を書くプロセス自体が、最高の説明力トレーニング**

#### この説明力が活きる場面
- 営業
- プレゼン
- 採用面接
- 取引先との交渉
- あらゆる場面で威力を発揮

**ビジュアル要素**:
- Before/After の吹き出し比較
- 説明力が活きる場面のアイコン

### セクション4: 社員・パートナーへの浸透効果
**見出し**: 理念が「お題目」から「行動指針」に変わる

**内容**:

#### 経営者の悩み
- 朝礼で話す、ミーティングで伝える、社内報に書く
- でも、なかなか浸透しない
- なぜか？ → **あなた自身が、理念を明確に言語化できていないから**
- 曖昧な理念は、曖昧にしか伝わらない

#### 本が解決する
- 本を書くことで、理念が明確に言語化される
- その本を社員に渡せば、一瞬で理念が伝わる

#### 実例: 製造業の場合
- 新入社員全員に自著を配布
- 入社3ヶ月後の理解度テスト: 30% → 90%
- 社員が自分の言葉で説明できるように
- 理念が「お題目」ではなく「行動指針」として機能

#### 採用面接での威力
- 「うちの会社は、こういう理念でやっています」と言いながら自著を手渡す
- 応募者の目が変わる:「この社長、本気だ」「この会社、信頼できる」
- 事例: IT企業で内定承諾率1.5倍

#### 社内教育ツールとして
- 新人研修で使う
- 管理職研修で使う
- 営業マニュアルの代わりに使う
- 繰り返し読まれ、組織全体に浸透

**ビジュアル要素**:
- 理念浸透度のグラフ（30% → 90%）
- 本を渡すシーンのイラスト
- 活用シーン（研修/採用/教育）のアイコン

### セクション5: 事業の見直しと改善につながる
**見出し**: 書こうとすると、矛盾や曖昧さが浮き彫りになる

**内容**:

#### 書く過程で気づくこと
- 「あれ、この部分、説明しづらいな」
- 「なぜこのサービスをやっているんだっけ？」
- 「この強み、本当に強みなのか？」
- **説明しづらい部分 = 事業そのものに問題がある可能性**

#### 実例1: 建設会社
- 書く過程で気づいた:「何でもやります」は強みになっていない
- むしろ弱みだった
- 思い切って「住宅リフォーム専門」に特化
- 結果: 説明がシンプルに、集客が楽に、利益率アップ

#### 実例2: 税理士事務所
- 書く過程で「自分が本当に提供したい価値は何か」が明確に
- 「記帳代行中心」→「経営コンサルティング中心」にシフト
- 結果: 売上変わらず、利益率2倍

#### 戦略の再構築
- 本を書くこと = 自分の事業を客観的に見つめ直す機会
- 「今のやり方で本当にいいのか？」
- 「もっと効率的な方法はないか？」
- 「次のステージに進むには何が必要か？」
- 書きながら、自然とこうした問いが浮かぶ

#### 次のステージへの道筋
- 本を書き終えた時、多くの経営者が感じること
- 「次に何をすべきか、はっきり見えてきた」
- 手元には、あなたの事業の「設計図」ができている

**ビジュアル要素**:
- 事業改善のプロセス図
- 実例の数字（利益率2倍など）を強調
- 設計図のメタファー画像

### セクション6: なぜ本を書くと事業理解が深まるのか
**見出し**: 3つの科学的根拠

**内容**:

#### 理由1: 「読者に伝える」という制約が思考を整理する
- 普段: 「自分がわかっていればいい」前提
- 本を書く時: 「初めて聞く人にもわかるように」という制約
- この制約が思考を整理する
- 曖昧な部分は曖昧なままでは書けない
- 論理の飛躍があれば、読者は理解できない
- だから、書く過程で自然と思考が磨かれる

#### 理由2: 構造化された情報は記憶に残りやすい
- 本 = 章立て、見出し、構造化
- 構造化こそが記憶に残る鍵
- バラバラの情報は忘れやすい、体系化された情報は忘れにくい
- 本を書くことで、事業に関する知識が体系化
- あなた自身の脳に深く刻まれる

#### 理由3: アウトプットが最強のインプット
- 学習科学の研究: インプットよりアウトプットの方が学習効果が高い
- 本を読むだけより、本を書く方が圧倒的に学びが深い
- 本を書くこと = 最高のアウトプット
- だからこそ、事業理解が驚くほど深まる

**ビジュアル要素**:
- 3つの理由をアイコン+図解
- 学習効果の比較グラフ（インプット vs アウトプット）

**見出し**: まずは3つを書き出してみてください

**内容**:

  1. あなたの事業の強みを3つ
  2. お客様が本当に求めているものは何か
  3. なぜこの事業を続けているのか

- **問いかけ**:
  - 書き出してみて、すぐに言葉にできましたか？
  - もし迷ったなら、本を書くプロセスが必ずあなたの助けになります

- **次のステップ**:
  - 事業の本質を言語化したい
  - 社員に理念を伝えたい
  - 次のステージへの道筋を見つけたい
  - そんなあなたは、まず無料相談から始めましょう

- **CTA**: 無料相談を申し込む

**ビジュアル要素**:
- 3つの質問を書き出すワークシートデザイン
- CTAボタン

## 🎯 期待される効果

### 訪問者の心理変化
1. **共感**: 自分も事業をうまく説明できない
2. **気づき**: 言語化できていないことが問題だったのか
3. **興味**: 本を書くと、こんなに深く理解できるのか
4. **納得**: 科学的根拠もあり、実例もある
5. **決意**: 自分も事業の本質を明確にしたい
6. **行動**: まずは無料相談で相談してみよう

### 差別化ポイント
- 「権威性」「営業力」ではなく、**内面的な変化**にフォーカス
- ストーリーベースで共感を呼ぶ
- 科学的根拠（学習科学）で納得感を高める
- すぐ試せるワークで行動を促す
